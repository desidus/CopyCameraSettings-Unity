# CopyCameraSettings-Unity
Plugin for Unity 2019.1 that allows to copy the settings of any Camera component to the Scene view Camera.

## How to use
Right click on any Camera component and select the desired command from the context menu.

[]https://i.imgur.com/R343Fei.png[/img]

## Requirements
Unity 2019.1 or newer.
